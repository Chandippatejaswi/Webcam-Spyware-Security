This project was developed by Shriya Banala, Sanjana Veeramally, Tejaswi Chandippa, SaiKiranSagar Koyeti, Shivakumar Kotra as part of a Cyber Security Internship. It is designed to secure organizations from webcam spyware attacks performed by hackers and ensures no spyware or malicious program secretly accesses your webcam without your knowledge.

Overview

This application protects user privacy by preventing unauthorized webcam access. It integrates:

  1. Face recognition for authentication
  2. Password + database user management
  3. Enable/Disable camera via system batch commands
  4. Intruder recording and SMTP alerting
  5. Activity logs and admin user management
